# Android Device Tree for Xiaomi Redmi A5 (serenity)

Este repositório contém a configuração básica (Device Tree) para o desenvolvimento de Custom ROMs (LineageOS) e Custom Recoveries (TWRP) para o Xiaomi Redmi A5.

## Especificações Técnicas
- **SoC**: Unisoc T606 (ums9230)
- **Arquitetura**: ARM64 (Cortex-A75/A55)
- **Kernel Header**: v4
- **Partição de Boot**: 64MB
